TEAM IMAGES README

Team Robot Images are placed in this directory in with names in the 
following format so they can be read:


Small Thumbnail used on multiple pages/listings: 
   team-{TEAM#}-small.jpg    (Example: team-3006-small.jpg)

Medium Thumbnail used on team information pages: 
   team-{TEAM#}-med.jpg    (Example: team-3006-med.jpg)

Other images of the robot (up to 5):
   team-{TEAM#}-{Image#}.jpg    (Example: team-3006-1.jpg)

These images are picked up by the teaminfo and teamdetails pages.
